# MyWebsite
个人网站展示

### Setting.json 配置
#### 基础信息
  - title: 网站名称
  - description: 网站描述
  - keywords: 网站关键词
  - author: 网站作者
  - qq: ~
  - email: ~
  - github: github地址
  - hitokotoUrl: 一言调用地址
  ```` 请前往 https://www.mxnzp.com/doc/list 申请 app_id 和 app_secret ````
  - app_id: 天气app_id
  - app_secret: 天气app_secret
  ```` netease: 网易云音乐; tencent: QQ音乐; kugou: 酷狗音乐; xiami: 虾米; kuwo: 酷我 ````
  - musicServer: netease服务
  ```` song: 单曲; playlist: 歌单; album: 唱片 ````
  - musicType: song服务
  - musicPlaylist: 歌单id
  - version: 版本
  - Copyright_text: 版权人
  - beian: 备案号

##### 第一屏
  - logo_img: logo图片
  - logo_text_1: 名称1
  - logo_text_2: 名称2
  - change1：名言
  - socialList: 社交链接列表
  - linkList: 首页卡片列表

##### 第二屏
  - logo_text_small: 名称
  - change2：名言
  - towLinkList: 次屏卡片列表

#### 设置
  - wallpapers: 壁纸
  - uptexts: 更新
